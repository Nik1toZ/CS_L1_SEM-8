## Задача

Бинарная классификация изображений солнечных панелей: определить, запылена поверхность панели или нет (классы `Clean` и `Dusty`). Запылённость снижает выработку электроэнергии на 10-30%, поэтому автоматическая детекция загрязнений помогает планировать обслуживание и сокращать потери на солнечных электростанциях.

## Датасет

Solar Panel Dust Detection: https://www.kaggle.com/datasets/hemanthsai7/solar-panel-dust-detection

-   2562 изображения
-   Clean: 1493, Dusty: 1069 (дисбаланс ~1.4:1)
-   Train / Val / Test: 70% / 15% / 15%

Скачивание датасета происходит автоматически внутри ноутбука через `kagglehub`.

## Метрики

Accuracy, Precision, Recall, F1-Score. Основная - F1. Recall критичен: пропуск запылённой панели обходится дороже ложной тревоги.

## Модели

Бейзлайн (`torchvision`, с предобученными весами ImageNet):

-   ResNet-18 - свёрточная сеть
-   ViT-B/16 - трансформер

Самописные (обучение с нуля):

-   CustomResNet18 - ручная реализация ResNet-18 (BasicBlock + 4 стадии остаточных блоков)
-   CustomViT - ручная реализация ViT-B/16 (patch embedding, CLS-токен, позиционное кодирование)

Каждая модель обучается в двух конфигурациях: базовой и с улучшениями.

## Результаты

Ключевые значения:

-   Бейзлайн ResNet-18: Accuracy 0.8776, F1 0.8498
-   Бейзлайн ViT-B/16: Accuracy 0.8880, F1 0.8716
-   Улучшенный ResNet-18: F1 0.8649
-   **Улучшенный ViT-B/16 (лучшая модель): Accuracy 0.8984, F1 0.8777**
-   CustomResNet18 + Aug (лучшая самописная): Accuracy 0.7917, F1 0.7546

Гипотезы улучшенного бейзлайна (аугментации, Label Smoothing, Weight Decay) подтвердились для обеих предобученных моделей. Самописные модели уступают ~0.10-0.12 по F1 - разрыв объясняется отсутствием предобучения, а не ошибками реализации (число параметров самописных совпадает с версиями из torchvision).
